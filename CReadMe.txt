README file for the C version  for CDS Assignment 2 2014
  v1.0 02/10/14

Compiling on CS Student system:
  You should not need this zip file. 

Running on Other Platforms. Assumption GNU gcc is available
  Unzip the zip file in the same directory as your program (distBakery.c).
  Then compile your executable by:
      gcc -Wall -o distBakery distBakery.c bakery.a

  Your program will be tested against the versions on the CS Student system
  and you must ensure that it works there before submission.
